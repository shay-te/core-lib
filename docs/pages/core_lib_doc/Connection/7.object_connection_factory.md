`ObjectConnectionFactory` wraps any plain Python object in Core-Lib's `with conn.get() as obj:` pattern. Use it when you have a data source that isn't a database — an in-memory store, a third-party client, or a test stub — but you still want consistent connection handling across your `DataAccess` layer.

*core_lib.connection.object_connection_factory.ObjectConnectionFactory* [[source]](https://github.com/shay-te/core-lib/blob/master/core_lib/connection/object_connection_factory.py){:target="_blank"}

## Usage

```python
from core_lib.connection.object_connection_factory import ObjectConnectionFactory


# Wrap any object
my_store = {'users': {}}
factory = ObjectConnectionFactory(my_store)

with factory.get() as store:
    store['users'][1] = {'name': 'Alice'}
```

With callbacks for lifecycle control:

```python
def on_open(obj):
    obj.connect()
    return obj

def on_close(obj):
    obj.disconnect()

factory = ObjectConnectionFactory(client, new_session_callback=on_open, close_session_callback=on_close)

with factory.get() as client:
    client.do_something()
# on_close is called automatically on exit
```

# ObjectConnectionFactory

*core_lib.connection.object_connection_factory.ObjectConnectionFactory* [[source]](https://github.com/shay-te/core-lib/blob/master/core_lib/connection/object_connection_factory.py){:target="_blank"}

## Functions

### \_\_init\_\_()

```python
def __init__(self, obj, new_session_callback=None, close_session_callback=None):
```

**Arguments**

- **`obj`**: The object to wrap.
- **`new_session_callback`**: Default `None`. Called with `obj` before each `get()`. The return value is what gets yielded inside the `with` block. Use this to clone, connect, or prepare the object per session.
- **`close_session_callback`**: Default `None`. Called with the session object on `__exit__`. Use this to disconnect or clean up.

### object()

Returns the original wrapped object directly, bypassing session lifecycle.

```python
@property
def object(self):
```

### get()

Returns an `ObjectConnection` context manager.

```python
def get(self, *args, **kwargs) -> ObjectConnection:
```

**Returns**

*`(ObjectConnection)`*: Context manager that yields the object (or the result of `new_session_callback`) on `__enter__`, and calls `close_session_callback` on `__exit__` if provided.

# ObjectConnection

class ObjectConnection(Connection):
