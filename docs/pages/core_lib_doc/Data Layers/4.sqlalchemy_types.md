SQLAlchemy's built-in column types don't cover everything. Core-Lib ships two custom types for common patterns: storing Python enums as integers, and storing geographic coordinates as PostGIS `POINT` geometry.

---

# IntEnum

*core_lib.data_layers.data.db.sqlalchemy.types.int_enum.IntEnum* [[source]](https://github.com/shay-te/core-lib/blob/master/core_lib/data_layers/data/db/sqlalchemy/types/int_enum.py){:target="_blank"}

Stores a Python `enum.Enum` as an integer in the database. On read, the integer is converted back to the enum member. On write, the enum member's `.value` is stored.

```python
class IntEnum(sa.types.TypeDecorator):
    impl = sa.Integer

    def __init__(self, enumtype, *args, **kwargs):
```

**Arguments**

- **`enumtype`**: The Python `enum.Enum` class to map to/from.

## Usage

```python
import enum
from sqlalchemy import Column, Integer
from core_lib.data_layers.data.db.sqlalchemy.base import Base
from core_lib.data_layers.data.db.sqlalchemy.types.int_enum import IntEnum


class User(Base):
    __tablename__ = 'user'

    class Role(enum.Enum):
        ADMIN = 1
        EDITOR = 2
        VIEWER = 3

    class Status(enum.Enum):
        ACTIVE = 1
        INACTIVE = 0

    id = Column(Integer, primary_key=True, nullable=False)
    role = Column(IntEnum(Role), default=Role.VIEWER)
    status = Column(IntEnum(Status), default=Status.ACTIVE)
```

Writing and reading:

```python
with db.get() as session:
    user = session.query(User).get(user_id)
    print(user.role)           # User.Role.ADMIN  (not 1)
    print(user.role.value)     # 1
    print(user.role.name)      # 'ADMIN'
```

# Point

*core_lib.data_layers.data.db.sqlalchemy.types.point.Point* [[source]](https://github.com/shay-te/core-lib/blob/master/core_lib/data_layers/data/db/sqlalchemy/types/point.py){:target="_blank"}

A SQLAlchemy custom type for storing geographic coordinates as a PostGIS `POINT`. Handles the conversion between Python dicts `{'latitude': ..., 'longitude': ...}` and the database `POINT(lon lat)` representation.

```python
class Point(UserDefinedType):
```

## Column definition

```python
from sqlalchemy import Column, Integer
from core_lib.data_layers.data.db.sqlalchemy.base import Base
from core_lib.data_layers.data.db.sqlalchemy.types.point import Point

class Location(Base):
    __tablename__ = 'location'

    id = Column(Integer, primary_key=True)
    coordinates = Column(Point())
```

## Static Methods

### to_point_str()

Converts `longitude` and `latitude` floats to a PostGIS `POINT` string for storing in the database.

```python
@staticmethod
def to_point_str(longitude: float, latitude: float, latitude_first: bool = False) -> str:
```

**Arguments**

- **`longitude`** *`(float)`*: Longitude value.
- **`latitude`** *`(float)`*: Latitude value.
- **`latitude_first`** *`(bool)`*: Default `False`. When `True`, emits `POINT(lat lon)` instead of `POINT(lon lat)`.

**Example**

```python
from core_lib.data_layers.data.db.sqlalchemy.types.point import Point

point_str = Point.to_point_str(longitude=34.7818, latitude=32.0853)
# 'POINT(34.7818 32.0853)'
```

### from_point_str()

Parses a PostGIS `POINT` string back to a `{'latitude': ..., 'longitude': ...}` dict.

```python
@staticmethod
def from_point_str(point_str: str, latitude_first: bool = False) -> dict:
```

**Arguments**

- **`point_str`** *`(str)`*: PostGIS `POINT` string, e.g. `'POINT(34.7818 32.0853)'`.
- **`latitude_first`** *`(bool)`*: Default `False`. Set to `True` if the point was stored with latitude first.

**Returns**

*`(dict)`*: `{'longitude': float, 'latitude': float}` (or reversed if `latitude_first=True`).

**Example**

```python
location = Point.from_point_str('POINT(34.7818 32.0853)')
# {'longitude': 34.7818, 'latitude': 32.0853}
```

### from_point_wkb()

Converts a WKB (Well-Known Binary) geometry object — as returned by GeoAlchemy2 — to a lat/lon dict.

```python
@staticmethod
def from_point_wkb(point) -> dict:
```

**Arguments**

- **`point`**: A WKB geometry object returned by a SQLAlchemy query when using `geoalchemy2`.

**Returns**

*`(dict)`*: `{'latitude': float, 'longitude': float}`.

**Example**

```python
with db.get() as session:
    location = session.query(Location).get(location_id)
    coords = Point.from_point_wkb(location.coordinates)
    # {'latitude': 32.0853, 'longitude': 34.7818}
```
